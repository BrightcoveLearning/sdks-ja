概要

Brightcoveのネイティブ SDK には、Android、iOS、および tvOS が含まれます。これらは、次のフレームワーク上に構築されています。

  • Android -上に構築されたライブラリAndroidExoPlayerプロジェクト（ExoPlayer 2 v2.9.3） [1]
  • iOS/tvOS - アップルAVFoundationフレームワークの上に構築されたライブラリ

特に指定がない限り、表の「はい」は、Native SDKによって現在サポートされているすべてのOSバージョンで機能がサポートされていることを示します。サポートされている OS バージョンのリストについては、以下を参照してください。

  • Android 用ネイティブ SDK 対応バージョン
  • iOS 対応バージョンのネイティブ SDK
  • tvOS 対応バージョンのネイティブ SDK
