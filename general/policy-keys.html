はじめに

ポリシーキーは、再生 API を使用して Video Cloud ライブラリからビデオおよび再生リストのコンテンツを取得するために使用されます。ポリシーキーはプレーヤーに設定されているため、プレーヤーレベルでコンテンツへのアクセスを制限できます。Video Cloud Studio Media モジュールを使用して、ビデオレベルでコンテンツの可用性を制御することもできます。

Brightcove Player を作成すると、各プレーヤーに対してポリシーキーが自動的に生成されます。このキーは、Brightcove アカウントに関連付けられ、動画が Video Cloud に保存されるという条件で、視聴できる動画と視聴できるタイミングを制御します。たとえば、ポリシーキーは、以下に基づいてアクセスを制限するために必要です。

  • アカウント
  • ドメイン制限 (使用する場合)
  • 地理的制限 (使用する場合)

ポリシーキーの詳細については、「概要」を参照してください。ポリシー API ドキュメント。

注意:
  • ポリシーキーは、Video Cloud に保存されている動画を制御するためにのみ使用されます。スタンドアロンの Brightcove Player を使用している場合は、ポリシーキーにアクセスしたり、取得したりする必要はありません。
  • 新しい Brightcove プレーヤー用に生成されたポリシーキーは、デフォルトでは検索が有効になっていません。再生 API から動画を検索するには、ポリシー API を使用して、検索が有効な新しいポリシーキーを作成する必要があります。詳細については、「ポリシー API の概要」ドキュメントを参照してください。

ポリシーキーの取得

ポリシーキーは自動的に生成され、管理されます。したがって、ほとんどの場合、それを要求したり管理したりする必要はありません。ポリシーキーを取得する最も簡単な方法は、次の操作を行うことです。

  1. PLAYERS モジュールを開き、新しいプレーヤーを作成するか、プラグインを追加するプレーヤーを見つけます。
  2. プレーヤーのリンクを選択して、プレイヤーのプロパティを開きます。

  3. 左側のナビゲーションメニューで、[ JSON エディタ ] を選択します。

  4. 表示されたJSONコードに、ポリシーキーが表示されます。

まれに、Brightcove Player を作成していない場合でも、ポリシーキーが必要になることがあります（Brightcoveのネイティブプレーヤ SDK を使用して厳密に開発する場合など）。この場合、カスタマーサポートに連絡してポリシーキーを取得する必要があります。

ポリシーキーの使用

ポリシーキーを使用して、Playback API へのリクエストを認証できます。Playback API は、一般的なメディア管理や他のシステムとの統合のために、Web ページやモバイルアプリで実行時に使用するためのものです。詳細については、再生 API インデックスページを参照してください。

ポリシーキーと API 認証情報

ポリシーキーは再生 API へのアクセスに使用されますが、API 認証情報は他の Video Cloud API のコンテンツにアクセスするために使用されます。

ポリシーキー

ポリシーキーは、Video Cloud ライブラリアセットのクライアント側での取得に使用されます。詳細は次のとおりです。

  • 再生 API を使用してビデオ/プレイリストデータにアクセスする
  • 読み取り専用
  • 動画/プレイリストデータは、Playback API によってサーバーにキャッシュされます
  • ポリシー API を使用して、次の操作を実行します。
    • ポリシーキーを作成します。ポリシーキーを取り消すには、カスタマーサポートに連絡する必要があります。
    • ポリシーレベルでドメイン制限を設定し、地理的制限を設定します。ポリシーキーを使用するプレイヤーによって制限が適用されます。
また、使用することができますBrightcovePlayerのカタログメソッド VideoCloudライブラリからコンテンツを取得します。

API 認証情報

API 認証資格情報（クライアント認証情報とも呼ばれます）を使用すると、サーバー側のアプリケーションで Video Cloud ライブラリアセットを管理できます。詳細は次のとおりです。

  • Video Cloud API を使用してデータにアクセスします。使用するアカウントと API のクライアント認証情報が必要になります。
  • 読み取り、挿入、更新、削除
  • CMS API によってビデオ/プレイリストのデータがサーバーにキャッシュされない
  • Video Cloud の API 認証 UI を使用して API 認証情報を作成および管理する

ポリシーキーの管理

ポリシーキーは、次のように管理できます。

取得する

ポリシーキーを更新する方法はありません。Video Cloud アカウントの既存のプレーヤーからポリシーキーを取得するか、ポリシー API を使用してポリシーキーを作成できます。

取り消す

ポリシーキーを取り消すには、カスタマーサポートに連絡する必要があります。

プレーヤーが削除されても、ポリシーキーは取り消されません。つまり、プレーヤーからポリシーキーを使用してからプレーヤーを削除した場合でも、ポリシーキーを引き続き使用できます。
