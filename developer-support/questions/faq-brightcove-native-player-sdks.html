以下は、Brightcoveのネイティブプレーヤー SDK に関連するよくある質問の一覧です。ここに見えない質問がありますか？おれに知らせてくれ！

一般的な質問

クローズドキャプションはサポートされていますか？

Player SDKは、マニフェスト内のWebVTT（HLS）と、サイドカーのWebVTTおよび608-over-708キャプションをサポートします。サポートの詳細については、ビデオフォーマットとキャプションのドキュメントを参照してください。


シークバーをカスタマイズできますか？

対応Brightcove ネイティブプレーヤーコントロールを使用する場合は、シークバーなどの要素をカスタマイズできます。詳細については、『 Brightcove ネイティブプレーヤーコントロール』ドキュメントを参照してください。


カスタムコントロールでビデオをオーバーレイすることはできますか？

Player SDK では、コントロールの使用を強制しません。あなたはそれらを使用するかどうかを選択することができます。Brightcove ネイティブプレーヤーコントロールを使用する場合は、エレメントをカスタマイズできます。詳細については、『 Brightcove ネイティブプレーヤーコントロール』ドキュメントを参照してください。


10秒のff/rewビデオ機能を追加できますか？

Player SDK はシークをサポートしているため、この機能を実装できます。


Chromecastはサポートされていますか？

Brightcoveのネイティブ SDK は、ChromeCastに接続されたテレビへの動画のストリーミングをサポートしています。

  • Android デバイスの場合は、 Android 用ネイティブ SDK 用のキャストプラグインを使用します。
  • iOS デバイスの場合は、 iOS 用ネイティブ SDK 用のキャストプラグインを使用します。

プレイリストはサポートされていますか？

はい。Brightcoveのネイティブ SDK には、再生 API から動画または再生リストを取得するためのメソッドが用意されています。詳細については、以下を参照してください。

  • Android 用ネイティブ SDK の VideoListSampleApp
  • iOS 版ネイティブ SDK の再生サービスを使用した Brightcove アセットの取得

Android特有の

HLSをAndroidで動作させるにはどうすればよいですか？

Brightcoveの Android 用 Player SDK は、Android 4.1 以降の HLS 再生機能を提供します。


HLSeはどうですか？

Brightcoveの Android 用プレーヤー SDK は HLSE での再生をサポートしています。

iOS特有の

AirPlayはサポートされていますか？

iOS 用 Player SDK は、広告がない限り AirPlay をサポートしています。Apple の AirPlay では、Apple デバイスからコンテンツをワイヤレスでストリーミングできます。


Apple TVはサポートされていますか？

iOS 用 Player SDK の最新リリースは、Apple TV の tvOS オペレーティングシステムをサポートしています。
