  • プレイヤーが破壊された後もハートビートが続く問題を修正しました

SSAI プラグインの変更:

  • ネストされたVASTおよびVMAP拡張要素の解析サポートが追加されました

    拡張要素は、ネストの深さに関係なく、SSAIプラグインによって実行されないため、このサポートにより、任意の深さのネストされた拡張要素を使用したVASTおよびVMAPの解析が失敗しないことが保証されます。

  • VASTの解析サポートが追加されましたAdVerification要素

    AdVerification要素はSSAIプラグインによって実行されないため、このサポートにより、VASTの解析はAdVerification要素は失敗しません。

    加えて、 AdVerification要素はVAST4.0の機能であり、Brightcove AndroidSDKはまだVAST4.xを完全にはサポートしていません。この解析サポートは、Brightcove Native SDK forAndroidでは一般的にVAST4.xのサポートと見なされるべきではありません。
