概要

Video Cloudのお客様の場合、ほとんどの場合、Android用のBrightcove Player SDKを使用すると、アプリは自動的に登録されます。Brightcove Player のお客様は、アプリからアカウント ID を送信する必要があります。以下の詳細を参照してください。

ビデオクラウドのお客様

Brightcoveプレーヤーとを使用する場合カタログビデオを取得して再生するためのSDKに含まれるクラスを使用すると、アプリは自動的にBrightcoveに登録されます。また、分析データが収集され、Video Cloud Analytics に表示されます。

オーバーライドする場合BrightcoveVideoViewクラスを作成するか、Brightcoveプレーヤーとカタログを使用しない場合は、Video Cloud PublisherIDをVideoCloudAnalyticsに送信する必要があります。これは、setAccount() 分析メソッドを使用して行うことができます。これにより、このアプリのデータを Video Cloud Analytics で表示することもできます。

Analytics analytics = brightcoveVideoView.getAnalytics();
analytics.setAccount("123456789");
Video Cloud Analyticsへのデータ送信の詳細については、 BrightcoveAnalyticsの使用ガイド。

Brightcoveプレーヤーのお客様

Brightcoveプレーヤーのお客様は、アプリをBrightcoveに登録する必要があります。これを行うには、分析を使用してアカウントIDをBrightcoveに送信しますsetAccount() 方法。

Analytics analytics = brightcoveVideoView.getAnalytics();
analytics.setAccount("123456789");
 

アプリの登録方法がわかったので、メディアコンテンツの操作方法を学ぶ準備が整いました。
